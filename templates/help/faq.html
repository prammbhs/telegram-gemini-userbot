General Questions

Telegram Gemini Bot is a web application that allows you to create AI-powered bots for your Telegram account. These bots use Google's Gemini AI to engage in natural conversations in Telegram groups or direct messages. The bots run on your personal Telegram account (as a userbot), giving them more capabilities than traditional Telegram bots.

Unlike regular Telegram bots (which use the Bot API and have the word "bot" in their usernames), Telegram Gemini Bot runs as a userbot:

  • It uses your personal Telegram account to send messages
  • It can join any group you have access to without needing admin permission
  • It appears to others as if you are typing and sending messages yourself
  • It can read message history and understand context from past conversations

This allows for much more natural integration into group conversations.

We offer both free and premium tiers:

  • Free Tier: Create and run bots with limited usage (500 AI responses per month and a maximum of 30 minutes runtime per session)
  • Premium Tier: Starts at $9.99/month and includes unlimited AI responses, longer bot sessions, priority support, and more advanced features

You can also use your own Google Gemini API key, which may provide additional free quota based on Google's current offering.

Account & Setup

To verify your Telegram account:

  1. Go to your Profile page
  2. Click on "Verify Telegram" in the Telegram Verification card
  3. Enter your phone number (with country code)
  4. We'll send a verification code to your Telegram app
  5. Enter the code on the verification page

After verification, your account is ready to use with our bots. This process is secure and uses Telegram's official authentication system.

A Gemini API key allows you to access Google's Gemini AI model, which powers the intelligence of your bots. To get one:

  1. Visit the Google AI Studio
  2. Sign in with your Google account
  3. Go to the "API Keys" section
  4. Click "Create API Key"
  5. Copy the key
  6. Paste it in your profile or during the setup wizard

Using your own API key may give you additional free quota depending on Google's current offerings.

Yes, your phone number is used solely for authenticating with Telegram's official API. We:

  • Never store your phone number in plaintext
  • Use industry-standard security practices
  • Don't share your information with third parties
  • Don't use your number for marketing

The verification process uses Telegram's official authentication flow, the same one used by the Telegram app itself.

Using Bots

Creating a bot is simple:

  1. Go to your Dashboard and click "New Bot"
  2. Enter a name for your bot
  3. Specify the target group or chat (username, group name, or chat ID)
  4. Define the context/persona (this shapes your bot's knowledge and personality)
  5. Set a duration for how long the bot should run
  6. Toggle the learning mode on/off
  7. Click "Create Bot"

After creating the bot, you can start it from the Dashboard or bot details page.

See our detailed guide for creating your first bot →

Learning Mode allows your bot to improve over time by analyzing conversations and adapting its responses. When enabled:

  • The bot remembers successful response patterns
  • It learns from questions and answers to improve future responses
  • It tracks topic transitions to better follow conversation flow
  • It adapts to the communication style of group members

Learning Mode data is stored securely and is only used to improve your specific bot. You can disable this feature at any time.

Yes, you can significantly customize how your bot responds through the Context/Persona field when creating or editing a bot. You can define:

  • Personality traits: Friendly, technical, humorous, formal, etc.
  • Knowledge areas: Specify what topics the bot should be knowledgeable about
  • Response style: Brief replies, detailed explanations, casual conversational tone, etc.
  • Character role: Make it act as a specific character or professional

For example, you might define your bot as "a friendly cryptocurrency expert who explains concepts in simple terms and occasionally uses emojis."

The more specific your context, the more tailored your bot's responses will be.

Troubleshooting

If your bot isn't responding, check these common issues:

  1. Bot Status: Verify that the bot shows as "Active" on your dashboard
  2. Telegram Verification: Ensure your Telegram account is properly verified
  3. API Key: Check if your Gemini API key is valid by testing it in your profile
  4. Target Group: Confirm the group/chat name or ID is correct
  5. API Quota: Check if you've exceeded your free tier API usage or Gemini quota
  6. Internet Connection: Ensure your server has a stable internet connection

You can view detailed logs for your bot on its detail page, which can help identify specific errors.

See our troubleshooting guide for more details →

If you're seeing an "Invalid API Key" error:

  1. Go to your Profile page
  2. Check your Gemini API key for typos or extra spaces
  3. Use the "Test Key" button to validate it
  4. If your key is invalid, get a new one from Google AI Studio
  5. Make sure you're using a Gemini API key, not a different Google API key

You can also try using our provided API credits by removing your custom API key.

If you're having trouble with Telegram verification:

  1. Ensure you're entering the correct phone number with country code
  2. Check that you're using the same phone number associated with your Telegram account
  3. Make sure you're entering the verification code exactly as received in Telegram
  4. If you have two-factor authentication enabled on Telegram, you'll need to enter your 2FA password when prompted
  5. Try logging out of Telegram on other devices and then verify again

If issues persist, try the "Re-authenticate" option on the Telegram verification page or contact our support team.
