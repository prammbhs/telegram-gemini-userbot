# Icons for Telegram Gemini Userbot

This directory contains icons used in the UI.

Required icons:
- play.png - For the Start button
- stop.png - For the Stop button
- settings.png - For settings
- chart.png - For analytics
- file.png - For file operations
- link.png - For URL operations
- stats.png - For statistics
- brain.png - For learning features

You can download free icons from sources like:
- https://www.flaticon.com/
- https://icons8.com/
- https://material.io/resources/icons/

Make sure the icons are approximately 64x64 pixels in size.
